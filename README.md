### 🎲 Rodando aplicação

```bash
# Clone este repositório
$ https://github.com/Adoilson-Freitas/desafio-front-end.git

# Acesse a pasta do projeto no terminal/cmd
$ cd desafio-front-end

# Instale as dependências
$ yarn 

# Rode a aplicação
$ yarn start

```

- [React.js](https://pt-br.reactjs.org/)
